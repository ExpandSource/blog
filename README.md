# blog

## 첫 생성 시 관련 정보, 요구사항?
- blog 만들 것이다.
- hugo로 만든다.
- github에 repo로 배포하고 수정할 계획이다.
- theme 가장 많이 사용되는 것들 중에서 simple한 것으로 고를 것이다.
- 마크다운 형태로 편집되게 할 것이고, 쉽게 작성 수정 배포되게 할 것이다.
- 개인용. 오픈되어있는 형태로 할 것이다.
- 사용자 역시 hugo에 첫 걸음이고, 익숙하지 않아 배우면서 할 것이다.
- 블로그 역시 익숙하지 않다. 기록과 오픈도 있겠지만, 연습용이 될 수도 있다.
- 가능하다면 docker에 올려서 통째로 서버로 이전하는 것도 배우고 싶다.
- 만들면서 배우고 싶고, 어느 정도 제한되고 규격화된 템플릿을 만들고 싶다.
- 개인화된 md to html css 디자인을 만들고 싶다.
- 주제는 자유, 그렇지만 it, ai, 도구 활용등의 기록을 적지 않을까?
- web 규격에 잘 맞춰서 가능하면 검색 친화적이면 좋겠지.
- 하지만 ai나 크롤러가 마구 긁어가지 않는 형태면 좋겠어.
- 글쓴이의 저작권은 존중되어야겠지. 그런걸 보호할 수 있는게 들어가면 좋겠어.
